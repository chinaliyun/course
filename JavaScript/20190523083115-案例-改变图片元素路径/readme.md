# 案例-改变图片元素路径

要求：

1. 在页面中插入一个按钮
2. 在页面中插入一个图片元素
3. 点击按钮第一次，把图片元素路径改为下面第一张图片
4. 再点击一次，改为第二张图片，以此类推
5. 当图片已经是最后一张时，点击按钮把路径重新改成第一张图片，以此循环
