# 运算符的优先级

在 JS 中运算符和 CSS 样式一样是有优先级的，当多种运算符在一起的时候，先进行哪种运算，再进行哪种运算，都有固定的顺序。

下表中的优先级从上到下逐步降低：

| 优先级 | 运算类型   | 运算符        |
| ------ | ---------- | ------------- |
| 20     | 圆括号     | `()`          |
| 17     | 后置递增   | `...++`       |
|        | 后置递减   | `...--`       |
| 16     | 逻辑非     | `!`           |
|        | 前置递增   | `++...`       |
|        | 前置递减   | `...++`       |
| 14     | 乘法       | `*`           |
|        | 除法       | `/`           |
|        | 取模       | `%`           |
| 13     | 加法       | `+`           |
|        | 减法       | `-`           |
| 11     | 小于       | `<`           |
|        | 小于等于   | `<=`          |
|        | 大于       | `>`           |
|        | 大于等于   | `>=`          |
| 10     | 等号       | `==`          |
|        | 非等号     | `!=`          |
|        | 全等号     | `==`          |
|        | 非全等号   | `!==`         |
| 6      | 逻辑与     | `&&`          |
| 5      | 逻辑或     | &#124;&#124;  |
| 4      | 三元运算   | `...?...:...` |
| 3      | 赋值运算符 | `=`           |
|        |            | `+=`          |
|        |            | `-=`          |
|        |            | `*=`          |
|        |            | `/=`          |
|        |            | `%=`          |
