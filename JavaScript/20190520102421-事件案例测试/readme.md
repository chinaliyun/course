# 事件案例测试

## 案例一

```
在页面中插入一个ul元素
在页面中插入一个button元素
给button元素添加一个点击事件
    每点击一次，创建一个li元素，元素内容设置点击事件发生的事件的时间戳
    把li元素插入到ul元素的结尾
```

## 案例二

```
在页面中插入一个ul元素
在页面中插入一个button元素
给button元素添加一个点击事件
    每点击一次，
    获取当前事件的时间，并格式化为2019-08-10 10:30:30的格式
    创建一个li元素，元素内容设置为上一步的时间
    把li元素插入到ul元素的结尾
```

## 案例三

```
在页面中插入一个ul元素
在页面中插入一个button元素
给button元素添加一个点击事件
    每点击一次，
    获取当前事件的时间，并格式化为2019-08-10 10:30:30的格式
    创建一个li元素，元素内容设置为上一步的时间
    把li元素插入到ul元素的结尾
当li元素的数量超过10个时，移除点击事件
```
