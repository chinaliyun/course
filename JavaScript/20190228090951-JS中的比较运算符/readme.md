# 比较运算符

比较运算符主要用于判断符号两边值的关系，返回一个**布尔值**，主要包括以下几种：

| 运算符 | 描述       | 例子      | 结果  |
| ------ | ---------- | --------- | ----- |
| `==`   | 等于       | `1=='1'`  | true  |
| `===`  | 绝对等于   | `1==='1'` | false |
| `!=`   | 不等于     | `1!='1'`  | false |
| `!==`  | 不绝对等于 | `1!=='1'` | true  |
| `>`    | 大于       | `1>2`     | false |
| `<`    | 小于       | `1<2`     | true  |
| `>=`   | 大于等于   | `1>=2`    | false |
| `<=`   | 小于等于   | `1<=2`    | true  |

比较运算符主要放在**条件语句**中使用，比如

```js
if (1 > 2) {
    alert("1>2");
} else {
    alert("1<2");
}
```
