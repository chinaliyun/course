# 案例-改变图片元素路径

1. 在页面中插入一个按钮
2. 在页面中插入 5 个图片元素
3. 所有图片元素，相对于 body 定位
4. 默认显示第 1 张图片，其他图片都隐藏
5. 点击按钮时，让当前显示的图片隐藏，让后一张图片显示
